\begin{abstract}
Este projeto teve como objetivo a identificação de duas espécies nativas da flora brasileira presentes no bioma Cerrado, o Buriti (\textit{Mauritia flexuosa}) e o Palmito Juçara (\textit{Euterpe edulis}), aplicando técnicas de \textit{Deep Learning}, mais especificamente o método RT-DETR. Para tal, foi utilizado um \textit{dataset} composto por imagens de alta resolução obtidas por meio de captura aérea com o uso de drone.
\end{abstract}
